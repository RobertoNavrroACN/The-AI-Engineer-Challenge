### Front End

This folder contains a minimal React + Vite frontend for the project.

Quick start:

1. Install Node.js (v18+ recommended).
2. From this folder run:

```bash
npm install
npm run dev
```

The dev server runs on port 3000 by default and proxies `/api` to `http://localhost:8000`.

Files added:

- `package.json` — project metadata and scripts
- `vite.config.js` — Vite config with React plugin and proxy
- `index.html` — app entry
- `src/main.jsx`, `src/App.jsx` — React entry and root component
- `src/api.js` — small API client
- `src/components/*` — `Header.jsx`, `Home.jsx`
- `src/styles.css` — basic styles
